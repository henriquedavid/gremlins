# GREMLINS
Gerenciador de Memória com Lista Encadeada Simples

# Compilação e Execução
- Na pasta raiz digite no terminal ```g++ -Wall -std=c++11 src/* -o driver -Iinclude -O3```. Para executar basta digitar ```./driver```.
